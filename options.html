Cómo obtener la URL del Web App

Sigue estos pasos para crear la hoja y el script que recibirán las pestañas guardadas:

  1. Crear la hoja
    Crea o abre una hoja en Google Sheets. Opcional: en la primera fila puedes poner los encabezados URL, Título y Fecha.
  2. Abrir Apps Script
    En esa hoja, ve al menú Extensiones → Apps Script. Se abrirá el editor del script.
  3. Pegar el código
    Borra el código que aparece por defecto en el editor de Apps Script y pega el siguiente código (selecciónalo todo y cópialo, o usa el botón «Copiar código»). Guarda el proyecto (Ctrl+S o el icono de disco). 
    function appendRowFromData(url, title, date) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  sheet.appendRow([url || '', title || '', date || new Date().toISOString()]);
}

function doPost(e) {
  try {
    var body = e.postData && e.postData.contents ? e.postData.contents : '{}';
    var data = JSON.parse(body);
    appendRowFromData(data.url || '', data.title || '', data.date || new Date().toISOString());
    return ContentService.createTextOutput(JSON.stringify({ ok: true })).setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService.createTextOutput(JSON.stringify({ ok: false, error: err.toString() })).setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  var params = e.parameter || {};
  var url = params.url || '';
  var title = params.title || '';
  var date = params.date || new Date().toISOString();
  var returnUrl = params.returnUrl || '';
  var errorMsg = '';
  try {
    appendRowFromData(url, title, date);
  } catch (err) {
    errorMsg = err.toString();
  }
  var html = '<!DOCTYPE html><html><head><meta charset="UTF-8"><meta name="viewport" content="width=device-width,initial-scale=1">';
  html += '<style>body{font-family:sans-serif;padding:2rem;text-align:center;} a{color:#1a73e8;} .err{color:#c5221f;}</style></head><body>';
  if (errorMsg) {
    html += '<p class="err">Error: ' + errorMsg.replace(/</g, '&lt;') + '</p>';
  } else {
    html += '<p><strong>Guardado en tu hoja</strong></p>';
  }
  if (returnUrl) {
    html += '<p><a href="' + returnUrl.replace(/"/g, '"') + '">Volver a la app</a></p>';
  }
  html += '</body></html>';
  return HtmlService.createHtmlOutput(html);
}
  4. Desplegar como aplicación web
    1. Haz clic en el botón azul «Implementar» (arriba a la derecha).
    2. En el menú, elige «Nueva implementación» (o «New deployment» si está en inglés).
    3. Donde dice «Seleccionar tipo» (o «Select type»), haz clic y elige «Aplicación web» («Web app»).
    4. En la ventana de configuración:
      • Descripción: Puedes dejar «Nueva implementación» o poner algo como «Guardar pestañas».
      • Ejecutar como: Yo (tu cuenta).
      • Quién tiene acceso: Cualquier persona (o «Anyone» en inglés).
    5. Pulsa «Implementar» (o «Deploy»).
  5. Autorizar la aplicación (solo la primera vez)
    1. Haz clic en «Autorizar acceso».
    2. Elige tu cuenta de Google.
    3. Si aparece «Google no ha verificado esta app», haz clic en «Avanzado» y luego en «Ir a [nombre del proyecto] (no seguro)».
    4. Acepta los permisos que pida.
  6. Copiar la URL
    Al terminar se mostrará la URL de la aplicación web (algo como https://script.google.com/macros/s/XXXXX/exec). Cópiala con el botón «Copiar» o seleccionándola, y pégala en el campo de arriba. Luego pulsa «Guardar».
